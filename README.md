# TobyLearn
A simple learning helper made with tauri as an alternative to quizlet.

## Features
- Import sets
- Export sets
- Create sets with GUI
- Edit sets with GUI
- Learn sets
- Multiple themes
- Basic settings (Case sensitivity, Randomize questions, ...)

## Planned features
- More settings
- Drag and drop sets
- Accent buttons
